

Что нужно:

1. node.js  
2. Сервер MySQL или любая длугая база данных
3. Клиентское приложение


Ход разработки:

1. Создаём папку и инициируем в ней npm
npm init --yes
2. Создаём файл index.js
3. Подключаем http и создаём сервер
4. Создаём роутер
4. Создаём папку controllers и в ней наши контроллеры
